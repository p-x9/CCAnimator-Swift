# CCAnimator

(Tested on iOS 13-14)

## Feature
### Custom UI
- Item Border Color
- Item Border Width
- Item Background Color
- Item Tint Color
- Item Corner Radius
- CC Background Color
  
### Animation
- Animation Kind
- Auto Reverse
